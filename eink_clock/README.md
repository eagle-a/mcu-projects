# ESP32-C3 电子墨水屏桌面时钟

## 项目概述

这是一个基于ESP32-C3和GDEY042T81 4.2寸电子墨水屏（400x300分辨率）的桌面时钟项目。项目通过WiFi连接网络，使用NTP协议同步时间，并在电子墨水屏上显示当前时间和日期。

## 硬件清单

- **MCU**: ESP32-C3（CORE ESP32C3开发板）
- **显示屏**: GDEY042T81 4.2寸电子墨水屏（400x300）
- **电源**: USB 5V供电（临时方案）
- **转接板**: 电子墨水屏转接板
- **外壳**: 桌面时钟外壳

## 屏幕规格

- **型号**: GDEY042T81
- **分辨率**: 400 x 300 像素
- **类型**: 黑白电子墨水屏
- **控制器**: IL0398
- **刷新模式**:
  - 全屏刷新: 约2秒（清除残影）
  - 快速刷新: 约1.5秒
  - 局部刷新: 无闪烁（用于时间更新）

## 引脚配置（基于CORE ESP32C3开发板）

根据开发板管脚定义，电子墨水屏接线如下：
- **BUSY（忙信号）**: IO08 (GPIO8) - 输入
- **RES（复位）**: IO05 (GPIO5) - 输出
- **DC（数据/命令）**: IO06 (GPIO6) - 输出
- **CS（片选）**: IO07 (GPIO7) - 输出
- **SCK（时钟）**: IO02 (GPIO2) - SPI2_CK
- **MOSI（数据）**: IO03 (GPIO3) - SPI2_MOSI

## 软件架构

### 组件说明

1. **电子墨水屏驱动** (`epd_driver.c/h`)
   - 从Arduino Display_EPD_W21驱动移植
   - SPI通信（SPI2_HOST，10MHz）
   - 支持全屏刷新、快速刷新、局部刷新
   - 深度睡眠支持

2. **WiFi模块** (`wifi.c/h`)
   - Station模式连接
   - 自动重连机制
   - 可配置最大重试次数

3. **RTC/NTP时间同步** (`rtc.c/h`)
   - SNTP协议同步网络时间
   - 支持多个NTP服务器
   - 本地时间转换
   - RTC保持时间

4. **时钟显示** (`clock.c/h`)
   - 时间格式化
   - 显示渲染
   - 边框绘制
   - 状态显示

5. **字体渲染** (`font.c/h`)
   - 大字体（16x32像素）- 显示时间
   - 中字体（12x24像素）- 显示日期
   - 小字体（8x16像素）- 显示状态
   - 居中显示功能

6. **主程序** (`main.c`)
   - 初始化序列
   - 任务管理
   - 显示更新循环
   - 深度睡眠电源管理

7. **配置文件** (`config.h`)
   - 集中管理所有可配置参数
   - WiFi账号密码
   - NTP服务器地址
   - 引脚定义
   - 刷新间隔
   - 深度睡眠开关

### 显示策略

- **初始显示**: 全屏刷新，绘制时钟边框
- **时间更新**: 每分钟更新一次（使用大字体显示时间，中字体显示日期）
- **定期全屏刷新**: 每5次更新进行一次全屏刷新，清除残影
- **电源管理**: 更新后进入深度睡眠（可在config.h中配置）

### 时间同步

- NTP服务器: pool.ntp.org, time.nist.gov
- WiFi连接后自动同步
- RTC保持时间（离线时仍可运行）

## 编译说明

### 前置要求

- ESP-IDF v5.0 或更高版本
- ESP32-C3开发板（CORE ESP32C3）
- GDEY042T81电子墨水屏及转接板

### 编译步骤

```bash
cd /home/zm/daily-dev/eink_clock
idf.py set-target esp32c3
# 编辑 main/config.h 配置WiFi账号密码
idf.py build
idf.py -p /dev/ttyUSB0 flash monitor
```

### 配置说明

编辑 `main/config.h` 文件配置以下参数：
- WiFi账号和密码
- NTP服务器地址
- 显示刷新间隔
- 引脚定义
- 深度睡眠开关

## 开发注意事项

### 电子墨水屏使用注意

1. 每次显示更新后必须进入深度睡眠模式，否则会缩短屏幕寿命
2. 通电时不要插拔电子墨水屏
3. 全屏刷新会有闪烁（正常现象）
4. 局部刷新无闪烁但会累积残影
5. 需要定期全屏刷新清除残影

### 电源优化

- 显示更新之间进入深度睡眠（在config.h中通过ENABLE_DEEP_SLEEP配置）
- 最小化WiFi活跃时间
- 优化刷新间隔
- 每次更新后屏幕进入深度睡眠以延长寿命

### 开发板注意事项（CORE ESP32C3）

1. **BOOT按键（IO09）**: 上电前不能下拉，否则ESP32会进入下载模式
2. **IO08（BUSY引脚）**: 下载烧录时为低电平，不能使用串口下载（我们只用它做输入，不影响）
3. **新款开发板**: GPIO18/19被USB占用，避免使用（本项目未使用）
4. **LED指示灯**: D4对应IO12，D5对应IO13（可用于调试）
